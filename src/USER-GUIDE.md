## Troubleshooting

### Theme not changing?
- Make sure you tapped a theme option
- The modal should close automatically
- Try closing and reopening the app

### Photos not appearing?
- Check camera permissions in device settings
- Make sure you have storage space
- Try taking a new photo instead of using gallery

### Settings not saving?
- Wait a moment after toggling
- Check that the switch moved position
- Try closing and reopening the app

### Progress not updating?
- Make sure you logged the activity
- Check that the save was successful
- Pull down on the screen to refresh
